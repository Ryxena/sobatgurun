# Docs

Struktur file

```text
/
├── app/                # Core application logic
│   ├── config/         # Configuration files (Database, App constants)
│   ├── controller/     # Controllers (Handle user requests)
│   ├── core/           # Framework core (App, Controller, Database wrapper)
│   ├── models/         # Database models
│   └── views/          # HTML Templates (Header, Footer, Pages)
├── public/             # Publicly accessible assets
│   ├── css/            # Stylesheets
│   ├── js/             # JavaScript files
│   ├── img/            # Images
│   └── index.php       # Entry point
```
